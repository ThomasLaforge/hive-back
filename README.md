# Projet initial pour vos exercices d'algorithmie en typescript

## Installation

1. Clonez le projet
2. Déplacez vous dans le dossier cloné
3. installez les dépendances avec la commande `npm install` 
4. lancez la synchro de la base de données avec la commande `npx prisma db push`
5. Lancez le projet avec la commande `npm start`
6. Modifiez le fichier `src/index.ts` pour commencer à coder ou directement dans le dossier des routes `src/routes`
